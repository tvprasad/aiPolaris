Azure AI Search hybrid query context for the Retriever node.

This file is intentionally minimal — the Retriever does not use an LLM.
It executes structured hybrid search (vector + keyword) against the AI Search index.

Search parameters (set in code, documented here for ADR-008 hash-pinning):
- query_type: semantic
- query_language: en-us
- semantic_configuration_name: default
- top: 5
- vector_fields: contentVector
- select: title, content, source, last_modified
- minimum_reranker_score: 0.60  (confidence threshold — ADR-002)
